AIger's Universe

AIger is an enterprise platform for prompt-to-workflow architecture, governed multi-agent execution, HITL approvals, A2A handoffs, source-grounded reports, and platform-aware copilot guidance.

Current Platform Shape

Workflow Builder
Prompt intake, clarifying questions, market validation, marketplace matching, generated agents, ReactFlow canvas, and final slide-panel review.
Run Console
Live SSE status, animated edges, compact timing estimates, HITL navigation, A2A messages, and final report.
AIger Copilot
Platform, KB, and general reasoning modes with files, tools, logs, citations, and collapsed tool activity.
Evidence Layer
Reusable KB documents, run-scoped workflow inputs, full-source citation viewers, and highlighted matched text.

Orchestrator Contract

  1. Understand the user prompt.
  2. Ask necessary questions inside the orchestrator log.
  3. Validate market signal with citations when tools are configured.
  4. Identify differentiators.
  5. Create Markdown technical design documentation.
  6. Plan agents, prompts, tools, frameworks, inputs, outputs, and HITL gates.
  7. Install exact marketplace matches through inline log approvals.
  8. Generate custom agents when no exact seed exists, then show the final slide panel.
